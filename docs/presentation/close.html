Ethics, continuous data, and your delivery

Serious buyers — and professors — reward candour. We state limits, avoid autonomous-betting hype, and show how today’s plumbing unlocks tomorrow’s daily data feeds.

Model card (honest limits)

What we don’t claim

No in-match dynamics, no head-to-head depth, no injury model. Performance can drift as the tour evolves; hard-court heavy training may bias clay/grass edge cases. Positioning: decision support — combine outputs with domain knowledge and risk policy. Especially relevant for betting-adjacent storytelling.

Roadmap: “daily data” without vaporware

  • Scheduled pipeline — retrain when new JeffSackmann seasons drop (cron / GitHub Actions).
  • Artifact lineage — every promoted model tied to a W&B run + Git SHA.
  • Drift awareness — monitor rank/surface distributions before silent degradation.
  • More consumers — anything that can POST JSON can use /predict at scale.
